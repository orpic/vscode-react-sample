![todo demo](/media/todo_demo.gif)

## Getting Started

```bash
# clone repo
gh repo clone orpic/vscode-react-sample

# navigate to repo
cd vscode-react-sample

# install deps
npm install
```

**Frontend**

```bash
# run in terminal
npm start
```
